# pruefung-app
Prüfungstool für die Wirtefachprüfung Basel-Stadt, kann im Browser per link (https://tcws57.github.io/pruefung-app/) aufgerufen werden.

Kann auf IOS Geräten zum Homescreen hinzugefügt werden und als Webapp ohne lästige Safari UI genutzt werden.

Die Fragen wurden dem Übungsfragenkatalog des Kantons Basel Stadt entnommen. (https://www.bs.ch/bvd/bgi/wirtefachpruefung)

Da ich das PDF nicht als besonders gutes Lernmittel empfinde, habe ich dieses Tool gebaut. 

Gruss und viel Spass beim lernen
